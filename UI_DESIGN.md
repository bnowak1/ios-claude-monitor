# Claude Monitor iOS App - UI Design Specifications

## Design Principles

1. **Glanceable**: Key info visible immediately
2. **Real-time**: Live updates without manual refresh
3. **Minimal**: No clutter, focus on what matters
4. **Dark-first**: Matches developer workflows
5. **Adaptive**: iPhone and iPad optimized

### Session Detail Components

**Header**:
- Back button + Project name
- Live indicator (pulsing when active)

**Session Info Card**:
- Machine name
- Full project path
- Session duration + model

**Stats Bar**:
- Message count
- Tool call count
- Token usage
- Tap for detailed breakdown

**Tab Bar**:
- **Events**: All events chronologically
- **Tools**: Tool calls only
- **Prompts**: User prompts only
- **Stats**: Detailed token/cost breakdown

**Event Timeline**:
- Grouped by time
- "NOW" section for live events
- Auto-scroll to newest (toggleable)
- Tool events with expand/collapse
- Color-coded by event type
